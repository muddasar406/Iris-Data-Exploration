# Iris Dataset Exploration and Visualization

## Task Objective

The objective of this project is to explore and visualize the Iris dataset
using Python. The project demonstrates basic exploratory data analysis (EDA)
techniques including statistical summary and graphical visualization.

---

## Dataset Used

The dataset used is the **Iris Dataset**, a classical dataset in machine learning.

It contains:

- 150 samples
- 4 numerical features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
- 1 categorical feature:
  - Species (Setosa, Versicolor, Virginica)
